use tailwind cli with jit without vite 
---------------------------------------

create index.html, src/style.css, build/style.css

add the ff in index.html
	<link rel="stylesheet" href="build/style.css">

add the ff in src/style.css
	@tailwind base;
	@tailwind components;
	@tailwind utilities;

generate tailwind.config.js file
	npx tailwindcss init

add the ff in tailwind.config.js
	mode:"jit",
	purge:["./index.html",] // or "./*.html"

need internet access to run next command

npx tailwindcss ./src/style.css -o ./build/style.css -m -w

open index.html file in Chrome or open with Live Server VSCode extension for real-time development


+---------+
| NOTICE: |
+---------+
To see help info, 
	npx tailwindcss -h

To have tailwindcss intellisense,
	npm init -y
	npm install -D tailwindcss

To host, (ignore node modules, package.json, package-lock.json to store on GitHub), src/style.css
