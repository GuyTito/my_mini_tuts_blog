use tailwind cli with jit without vite 
---------------------------------------

create index.html, src/style.css, build/style.css

add the ff in index.html
	<link rel="stylesheet" href="build/style.css">

add the ff in src/style.css
	@tailwind base;
	@tailwind components;
	@tailwind utilities;

need internet access to run next command

generate tailwind.config.js file
	npx tailwindcss init

add the ff in tailwind.config.js
	mode:"jit",
	purge:["./index.html",], // or "./*.html"

npx tailwindcss -i ./src/style.css -o ./build/style.css -m -w

open with Live Server VSCode extension for real-time development or open index.html file with Chrome(make sure src and href paths are correctly linked to respective files)


+---------+
| NOTICE: |
+---------+
To see help info, 
	npx tailwindcss -h

To have tailwindcss intellisense,
	npm init -y
	npm install -D tailwindcss

Ignore node modules, package.json, package-lock.json, (to store on GitHub) and src/style.css to deploy.
